# RFID-Cam-DoorGuard
本项目是一个基于STM32F103CBT6微控制器的智能门禁系统
> 精简代码后可以达到200*200像素的图片传输，C8T6同样可以使用，但是只能稳定传输160*120的图片

| 模块           | 功能描述                     | 
|----------------|------------------------------|
| **STM32F103C8** | 主控芯片，协调各模块通信 |
| **MFRC522**     | RFID卡号读取与身份验证 
| **OV2640**      | 实时JPEG图像采集与传输|
| **ESP8266**     | Wi-Fi联网，远程指令收发 |
| **SG90舵机**    | 门锁驱动，支持0°~90°控制 |
